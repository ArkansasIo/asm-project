# Dragon Warrior NES Clone

## Project Structure

- main.asm: Main game logic
- map.asm: Map system
- battle.asm: Battle system
- menu.asm: Menu system
- data.asm: Game data
- graphics.asm: Graphics data
- sound.asm: Sound/music/SFX
- console_build.asm: NES console build routines
- editor_graphics.asm: 8-bit graphics editor
- editor_algo.asm: Graphics/sound creation algorithms
- editor_integration.asm: Editor integration
- editor_menu.asm: Editor menu system
- editor_tools.asm: Editor tools system
- editor_audio.asm: Editor audio system
- TODO.asm: Task list

## Folders

- src/dragonwarrior/: All game and editor ASM modules
- docs/: Documentation
- tools/: Emulator and utilities

## Build

Use ca65/ld65 to assemble and link the ROM.
