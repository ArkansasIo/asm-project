# Installing MinGW-w64 (Reference)

1. Download from https://www.mingw-w64.org/downloads/
2. Run the installer and select x86_64 architecture
3. Add the `bin` directory to your system PATH
4. Verify with `g++ --version` in a new terminal

This clone is for documentation and integration only; use the official MinGW-w64 for real builds.
