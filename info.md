# 8BitStudio Project Information

This project is a homebrew 8BitStudio ROM, written in 6502 assembly, with BIOS, CMOS RAM, splash/boot screens, menus, and an in-ROM editor.

## Features
- 8BitStudio ROM with simulated BIOS and CMOS
- Splash screen, boot screen, BIOS menu, submenus
- CMOS RAM screen
- In-ROM editor/tool (WIP)
- Modular assembly source files

## Folders
- src/: Assembly source code
- docs/: Design and planning documents

## Building
Use a 6502 assembler (e.g., NESASM, ca65) to build the 8BitStudio ROM from src/.

## 16-bit Version
A 16-bit version (for SNES or similar) is planned for future development.
